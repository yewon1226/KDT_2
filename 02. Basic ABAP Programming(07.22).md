### 1) 필드 심볼(Field Symbol)
- 문자열에 저장된 변수명을 `ASSIGN ( )` 으로 실제 변수에 연결하여 값을 변경
```abap
FIELD-SYMBOLS <필드심볼명> TYPE 타입.

ASSIGN 변수 TO <필드심볼명>.                          "일반 변수
ASSIGN (변수명) TO <필드심볼명>.                     "동적 변수명
ASSIGN COMPONENT 필드명 OF STRUCTURE 구조체 TO <필드심볼명>.      "구조체 필드
ASSIGN COMPONENT 변수 OF STRUCTURE 구조체 TO <필드심볼명>.        "동적 구조체 필드

LOOP AT 내부테이블 ASSIGNING <필드심볼명>.
  ...
ENDLOOP.
```
```abap
CLASS zcl_a06_w03 IMPLEMENTATION.
  METHOD if_oo_adt_classrun~main.
    DATA: lv_name(20),
          lv_carrid TYPE scarr-carrid.
    FIELD-SYMBOLS <f_name> TYPE any.
    lv_name = 'LV_CARRID'.
    ASSIGN lv_name TO <f_name>.
    <f_name> = 'AA'.

    out->write( lv_carrid ).
  ENDMETHOD.
ENDCLASS.
```

- `AT END OF` : 같은 값으로 묶인 데이터가 끝나는 시점(마지막 행)에서 실행되는 Control Break 문법
```abap
AT END OF <field>.
  ...
ENDAT.
```
```abap
CLASS zcl_a06_w04 IMPLEMENTATION.
  METHOD if_oo_adt_classrun~main.
    TYPES: BEGIN OF ts_data,
             month(2),
             mcount TYPE I,
           END OF ts_data,
           tt_data type table of ts_data.
    DATA: lt_data TYPE tt_data,
          lt_sum LIKE lt_data.
    lt_data = value #(
                        ( month = '01' mcount = '10') ( month = '02' mcount = '20') ( month = '03' mcount = '30') ( month = '04' mcount = '40')
                        ( month = '01' mcount = '20') ( month = '02' mcount = '40') ( month = '03' mcount = '30') ( month = '04' mcount = '40')
                    ).
    SORT lt_data BY month.

    LOOP AT lt_data INTO DATA(ls_data).
      AT END OF month.
        SUM.
        APPEND ls_data TO lt_sum.
      ENDAT.
    ENDLOOP.
    out->write(  lt_sum ).
  ENDMETHOD.
ENDCLASS.
```

```abap
CLASS zcl_a06ex02 DEFINITION
  PUBLIC
  FINAL

  PUBLIC SECTION.
    TYPES: BEGIN OF ts_temp,
             year      TYPE c LENGTH 4,

    DATA: lt_temp  TYPE tt_temp,

    TYPES: BEGIN OF ts_sbook,
             year   TYPE c LENGTH 4,
             carrid TYPE sbook-carrid,
             class  TYPE sbook-class,
             res01  TYPE i, can01   TYPE i,
             res02  TYPE i, can02   TYPE i,
             res03  TYPE i, can03   TYPE i,
             res04  TYPE i, can04   TYPE i,
             res05  TYPE i, can05   TYPE i,
             res06  TYPE i, can06   TYPE i,
             res07  TYPE i, can07   TYPE i,
             res08  TYPE i, can08   TYPE i,
             res09  TYPE i, can09   TYPE i,
             res10  TYPE i, can10   TYPE i,
             res11  TYPE i, can11   TYPE i,
             res12  TYPE i, can12   TYPE i,
           END OF ts_sbook.
    TYPES tt_sbook TYPE TABLE OF ts_sbook.

    METHODS set_book
      IMPORTING pa_carr TYPE sbook-carrid
      EXPORTING lt_book TYPE tt_sbook.
  PROTECTED SECTION.
  PRIVATE SECTION.
ENDCLASS.

CLASS zcl_a06ex02 IMPLEMENTATION.
  METHOD set_book.
    SELECT FROM sbook
        FIELDS substring( fldate, 1, 4 ) AS year, carrid, class, substring( fldate, 5, 2 ) AS month, cancelled
        WHERE carrid = @pa_carr
        INTO CORRESPONDING FIELDS OF TABLE @lt_temp.

    SELECT FROM sbook
        FIELDS DISTINCT substring( fldate, 1, 4 ) AS year, carrid, class
        WHERE carrid = @pa_carr
        INTO CORRESPONDING FIELDS OF TABLE @lt_book.

    SORT lt_temp BY year class month.
    SORT lt_book BY year class.

    DATA: lv_res TYPE i,
          lv_can TYPE i.

    LOOP AT lt_temp ASSIGNING FIELD-SYMBOL(<ls_temp>).
      IF <ls_temp>-cancelled = 'X'.
        <ls_temp>-can = 1.
        <ls_temp>-res = 0.
      ELSE.
        <ls_temp>-res = 1.
        <ls_temp>-can = 0.
      ENDIF.
    ENDLOOP.

    LOOP AT lt_temp INTO DATA(ls_temp).
      AT END OF month.
        SUM.
        APPEND ls_temp TO lt_temp2.
      ENDAT.
    ENDLOOP.

    FIELD-SYMBOLS: <fs_book>  TYPE any,
                   <fs_month> TYPE i.

    LOOP AT lt_temp2 INTO DATA(ls_temp2).
      READ TABLE lt_book ASSIGNING <fs_book>
        WITH KEY year   = ls_temp2-year
                 carrid = ls_temp2-carrid
                 class  = ls_temp2-class.
      IF sy-subrc = 0.
        DATA(lv_res_month) = |RES{ ls_temp2-month }|.
        ASSIGN COMPONENT lv_res_month OF STRUCTURE <fs_book> TO <fs_month>.
        <fs_month> = ls_temp2-res.

        DATA(lv_can_month) = |CAN{ ls_temp2-month }|.
        ASSIGN COMPONENT lv_can_month OF STRUCTURE <fs_book> TO <fs_month>.
        <fs_month> = ls_temp2-can.
      ENDIF.
    ENDLOOP.
  ENDMETHOD.
ENDCLASS.
```
